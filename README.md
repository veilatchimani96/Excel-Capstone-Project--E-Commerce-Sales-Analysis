# 📊 E-Commerce Sales Analysis Project

## 📌 Overview
This project focuses on analyzing an e-commerce sales dataset using **Excel and Power BI**. It includes data cleaning, data imputation, and interactive dashboard creation to generate meaningful business insights.

---

## 🎯 Objectives
- Clean and preprocess raw data
- Handle missing values using imputation techniques
- Build relationships between datasets
- Analyze sales performance across regions, products, and customers
- Create an interactive dashboard for insights

---

## 🧹 Data Cleaning
- Removed inconsistencies in **Customer ID, Product ID, and Country**
- Standardized customer names using `TRIM()` and `PROPER()`
- Ensured proper relationships between all tables
- Verified and corrected data types

---

## 🔧 Data Imputation
- Replaced missing **Loyalty Level** values with `"Unknown"`
- Filled missing **Cost** and **Stock** using sub-category averages
- Imputed missing **Sales values** (Quantity, Price, Discount) using averages
- Used `VLOOKUP()` to merge data across sheets

---

## 📊 Data Analysis
- Analyzed customer **Count of Region wise age**
- Compared **Month Wise Sales Trend**
- Evaluated **Sales by Category**
- Examined **Average Unite Price by Subcategory**
- Tracked **Gender distribution by Region**
- Analyzed **Revenue by Loyalty Tier**
  
---

## 💡 Key Insights

- 📍 Age groups vary by region → supports targeted marketing  
- 📈 Sales show seasonal monthly trends  
- 🛒 Few categories drive most revenue  
- 💲 Unit prices differ across subcategories  
- 👥 Gender mix varies by region  
- 🎯 High loyalty tiers generate more revenue
